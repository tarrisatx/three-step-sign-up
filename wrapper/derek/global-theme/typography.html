Typography

Every visual element of our brand says something about us — even down to the fonts we use and how we use them. The specific styles, sizes and weights communicate the essence of the brand and place our brand within a meaningful hierarchy.

Heading Usage

h1 - h3 can be used in $darkest-gray and $white only and all caps only.
They all use Open Sans $heading, uppercase .uppercase;, and font-weight of 200.

Page Title

h1. font-size: 2.25em; For use in lead areas only.

River heading

h2. font-size: 1.75em; For use as section, page or river headings.

Element heading

h3. font-size: 1.25em; For use as title or headings of elements. i.e. OfferBox, Value Props, Grid module elements, any grid style graphic element.

Subhead Usage

H4-h6 can be used in title case only.


They use Khand $subhead and a font-weight of 400.

Sub-heading

h4. font-size: 2rem; For use as a sub-heading above or beneath an h1 or h2. Only to be used in $gray-darkest or $white.

If used as accent to title it should be red and have no margin

Text Title

h5. font-size: 1.75rem; For use as a title or heading within a text section. Can be used in $gray-darkest.

Accent Text Title

h6

Body Copy

Body copy blocks should be left aligned by default. It should only center if the pattern centers it automatically.

Do not exceed more than two links per paragraph and links should not be more than four words long. Links outside of the paragraph tag but within the text block should be text links with the class .read-more, never buttons.
